# Практика алгоритмов и архитектур классического ML и DL

Репозиторий содержит набор Jupyter-ноутбуков с практическими примерами и тренировочными моделями в области классического машинного обучения (ML) и глубокого обучения (DL)

## Структура

- **`classic_ml/`** - ноутбуки, посвященные классическим алгоритмам машинного обучения:
  - Линейная и логистическая регрессия
  - Деревья решений и случайный лес
  - SVM, KNN и другие методы
  - Примеры предобработки данных и оценки моделей

- **`classic_dl/`** - ноутбуки по глубокому обучению:
  - Полносвязные нейронные сети (MLP)
  - Сверточные нейронные сети (CNN) для задач компьютерного зрения
  - Трансформенные нейронные сети
  - Автоэнкодеры
  - Графовые нейронные сети (GNN)
  - Использование фреймворка PyTorch

- **`projects/`** - мини-проекты: от анализа данных до развертывания или визуализации результатов
